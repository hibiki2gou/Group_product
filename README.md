# Training Log App (筋トレ記録管理システム)

このアプリケーションは、日々のトレーニング内容（種目、重量、回数、セット数）を詳細に記録し、データの可視化を通じて自身の成長を分析・サポートするための Flask ベースの Web アプリケーションです。

##  主な機能

* **詳細なワークアウト記録**: 
    * 日付ごとに種目、セット数、レップ数、重量を記録できます。
    * 各セットにコメントを残すことが可能です。
* **トレーニングデータの可視化**:
    * **日別積み上げ棒グラフ**: 部位ごとの合計負荷量（重量 × 回数 × セット）を日別・週別で確認できます。
    * **部位別パイチャート**: 全体のトレーニング量に対する各部位（胸、背中、脚など）の比率を可視化します。
    * **筋肉人形（部位別ステータス）**: 解剖図に基づいた直近のトレーニング傾向を確認できます。
* **カスタマイズ可能な種目設定**:
    * ユーザー自身で種目の追加、削除、並び替えが可能です。
    * 既存のログがある種目は「論理削除」され、過去の記録を保持したまま一覧から非表示にできます。
* **ユーザーレベルに応じた目標管理**:
    * 初心者・中級者・上級者の難易度設定により、自分に適した目標負荷量の倍率を設定できます。
* **マッスル格言機能**: 
    * ページ読み込み時に、著名なボディビルダーや偉人による格言がランダムに表示され、モチベーションを高めます。

##  使用技術

### バックエンド
* **Python 3.x**
* **Flask**: Web フレームワーク
* **Flask-SQLAlchemy**: データベース ORM (SQLite 使用)
* **Flask-Migrate**: データベースマイグレーション管理
* **Flask-WTF**: フォームバリデーション

### フロントエンド
* **JavaScript (ES6+)**
* **Chart.js / ChartDataLabels**: グラフ描画
* **HTML5 / CSS3**
