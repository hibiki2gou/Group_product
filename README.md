# MachoMachoMacho (筋トレ記録・分析プラットフォーム)

自身のトレーニング負荷をデータ化し、視覚的なフィードバックを通じて成長を実感するためのフルスタックWebアプリケーションです。

## 1. 概要
「データで筋肉を育てる」をコンセプトに、単なる数値記録を超えた分析機能を提供します。
過去のトレーニング負荷を計算し、SVG形式の筋肉図をヒートマップ化することで、現在の鍛え具合を直感的に把握することが可能です。

## 2. 主な機能

###  トレーニング分析・可視化
* **日別積み上げグラフ**: 過去7日間の総負荷量（セット×レップ×重量）を種目ごとに積み上げて表示。
* **部位別ヒートマップ**: 過去14日間の目標達成率に基づき、筋肉図がグレーから赤へ動的に色変化。
* **週次レポート**: 週ごとの合計重量推移と、部位別のボリューム比率を円グラフで可視化。
* **部位別詳細分析**: 特定部位の直近30日間の重量推移をバーチャートで確認。

###  記録・管理機能
* **スマート・ロギング**: 日付ごとに種目、セット、レップ、重量、コメントをクイックに記録。
* **カレンダー連携**: 記録済みの日は色分けされ、過去のログへ簡単にアクセス可能。
* **種目カスタマイズ**: ドラッグ感覚の並び替え、新規追加、論理削除に対応した種目管理。

###  ユーザー・目標設定
* **ダイナミック・ターゲット**: ユーザーのレベル（初級〜上級）やカスタム倍率に応じ、部位ごとの目標値を動的に算出。
* **BIG3ベスト記録**: ベンチプレス、デッドリフト、スクワットの過去最高重量を自動抽出して表示。

## 3. 技術スタック

### バックエンド
* **Python / Flask**: メインフレームワーク。
* **SQLAlchemy (ORM)**: オブジェクト指向なデータベース操作。
* **Flask-WTF**: セキュアなフォームバリデーション。
* **Flask-Migrate**: データベースのマイグレーション管理。

### フロントエンド
* **JavaScript (Vanilla JS)**: Fetch APIを用いた非同期通信と動的なDOM操作。
* **Chart.js**: データの多角的なグラフ描画。
* **SVG**: インタラクティブな筋肉図ヒートマップの実装。
* **Jinja2**: 効率的なテンプレート継承とデータ埋め込み。

### データベース
* **SQLite**: 軽量でポータブルなリレーショナルデータベース。

## 4. エンジニアとしてのこだわり（工夫ポイント）
* **論理削除の実装**: 種目を削除しても過去の統計データに影響を与えないよう、`is_deleted` フラグによる論理削除を採用。
* **シームレスなUX**: ページ遷移を抑えるため、種目の編集や記録の削除に非同期通信（AJAX/Fetch）を多用。
* **モチベーション設計**: ログイン時のランダム名言表示や、未実施種目への鼓舞メッセージなど、継続を促すUXを追求。
